# Custom Tagging Input - Angular AEM Test

Create a custom **tagging input** component that will work for both **reactive form** and **model**.

## Installation

Run as Angular project.

STEP 1
```bash
npm install
```
STEP 2
```bash
ng serve
```
[Live Demo](https://aem-test-5c1a4.firebaseapp.com/)
